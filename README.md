La aplicación cuenta con una pantalla inicial de Login/SignIn.

En esta pantalla se debe primero crear un usuario seleccionando el botón SIGN IN, se debe tener en cuenta que el campo correo debe tener un formato válido de correo y el campo password debe tener una longitud mínima de 5 caracteres, una letra mayúscula, una letra minúscula, un caracter especial (@$!%*?&#) y un número.

Una vez creado el usuario se puede realizar el ingreso con el botón LOGIN con el correo y la contraseña de un usuario creado previamente.

El botón de LOGIN redirecciona a la pantalla que muestra el listado de establecimientos consultado al servicio.
